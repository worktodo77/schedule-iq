- Report Card: `light_report.png`, `dark_report.png`
- Checks: `light_checks.png`, `dark_checks.png`

- Paths: `light_paths.png`, `dark_paths.png`
- Forensics: `light_forensics.png`, `dark_forensics.png`
- Settings: `light_settings.png`, `dark_settings.png`

Rebuild the review set from the repository root with:
